TradingView-Webhook-Bot

发表于 2021-05-15 分类于 网络技术

关于

TradingView Webhook Bot 可以让 TradingView 警报通过 webhooks 立刻通知到Telegram, Discord, Twitter或Email。

安装步骤

  1. 克隆此存储库 git clone https://github.com/jsjsjs667788/TradingView-Webhook-Bot.git

  2. 创建虚拟环境 python3 -m venv TradingView-Webhook-Bot

  3. 激活它 source TradingView-Webhook-Bot/bin/activate && cd TradingView-Webhook-Bot

  4. 安装依赖 pip install -r requirements.txt

  5. 修改 config.py

  6. 设置TradingView警报。警报消息的示例为：

    • key 是必填项！ 它必须和 config.py 里的sec_key 保持一致。
    • telegram, discord, slack 是可选的。根据你自己的需要去填写相对应的通知渠道。
    • msg 可以是任何内容。 对于 Telegram 和 Discord 支持Markdown语法
      • TradingViews 中像这样的变量 {{close}}, {{exchange}} 如何使用，请参照 这里 。
    • 你的 webhook 链接 为 http://<YOUR-IP>/webhook

  7. 如果使用防火墙，请确保打开相应的端口。

  8. 运行机器人 python main.py

*建议在其他端口（例如8080）上运行。然后将80端口映射到8080。*

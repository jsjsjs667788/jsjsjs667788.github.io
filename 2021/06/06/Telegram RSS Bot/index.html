Telegram RSS Bot

发表于 2021-06-06 更新于 2021-06-29 分类于 网络技术

关于

Telegram RSS Bot 是一个订阅类机器人，可以方便订阅需要的信息通知到Telegram。

安装步骤

  1. 安装docker

apt-get update && apt-get upgrade -y
apt-get install curl -y
curl -fsSL https://get.docker.com/ | sh
systemctl start docker
systemctl enable docker
systemctl status docker

  1. 运行容器
    docker run --name rssbot -d -v /var/data:/app/data/ -e RSSBOT_TOKEN=<123456:abcdef123456> fengkx/node_rssbot
  2. 自动更新容器
